# Persian Subtitle Fixes

Its a free software to enhance Persian subtitles.

**Features**
- Dialog Hyphen Fix
- Comma Fixes
- Dot Fixes
- Exclamation Mark Fixes
- Question Mark Fixes
- Remove Unneeded Spaces
- Parentheses Fixes
- Double-Quotes Fixes
- Dash Fixes
- Change Arabic Char to Persian
- Space to Invisible Space
- and more.
