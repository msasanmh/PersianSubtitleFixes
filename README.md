# Persian Subtitle Fixes

Its a free software to enhance Persian subtitles.

**Features**
- Change Arabic Chars to Persian
- Remove Unneeded Spaces
- Add Missing Spaces
- Fix Dialog Hyphen
- Fix Wrong Chars
- Fix Misplaced Chars
- Fix Abbreviations
- Space to Invisible Space
- Remove Leading Dots
- Remove Dot from the End of Line
- and more.
